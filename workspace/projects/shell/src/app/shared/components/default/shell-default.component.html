  • About the Apps structure and deployment

  • workspace
    • projects
      • shell (main) app
      • mf-a (remote) app
      • mf-b (remote) app
      • lk-common-lib (contains only one component AngularVersionComponent)
  • Deployment steps
    1. npm run remove-dists (dist, dist-gh)
    2. npm run build-all (npm run build-lib && npm run build-mf-a && npm run build-mf-b && npm run build-shell)
    3. npm run deploy-gh (npm run prepare-gh-pages && gh-pages -d dist-gh -b gh-pages)
  • Local Run (do not forget cd workspace)
    1. npm run build-lib-dev
    2. npm run start-b
    3. npm run start-a
    4. npm run start-s
    5. localhost:4200
